# Archive-35 CaFE Uploader

Chrome extension to upload photos from your CaFE Ready folder to your CallForEntry.org portfolio.

## Install

1. Open `chrome://extensions/` in Chrome
2. Enable **Developer mode** (toggle, top right)
3. Click **Load unpacked**
4. Select this `cafe-uploader/` folder
5. Pin the extension icon in your toolbar

## Usage

1. **Log in to CaFE** — open https://artist.callforentry.org and sign in
2. **Click the extension icon** in your toolbar
3. **Select your CaFE Ready folder** — the one with your prepped images + `cafe_metadata.csv`
4. **Review** — the extension shows what's new vs already uploaded
5. **Click "Upload All New"** — sit back while it uploads

## Requirements

- Chrome browser
- Active CaFE login session (the extension uses your existing cookies)
- CaFE Ready folder with:
  - JPEG images (< 5MB each)
  - `cafe_metadata.csv` or `submission.json` (generated by Studio CaFE tab)

## How It Works

The extension reads your local images + metadata file, then uses CaFE's existing upload form to submit each image with all fields pre-filled (title, description, dimensions, medium, year, price, discipline). It cross-references against your current CaFE portfolio to skip duplicates.

## Troubleshooting

- **"No CaFE Tab"** — Open artist.callforentry.org in a tab first
- **"Could not find form security tokens"** — Your CaFE session may have expired. Log in again.
- **Upload fails** — Check that images are JPEG, under 5MB. The Retry button will re-attempt failed uploads.
